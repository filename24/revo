# Revo.edu
Монголын боловсролын төвүүдийг нэгтгэх ухаалаг систем

## Асуудал
Монголд сурагчид болон эцэг эхчүүдийн хувьд:
- Хүүхдэдээ тохирох сургалтын төв хайх хэцүү
- Байршил, цагийн тохироо, үнэ зэрэг мэдээлэл тархай бутархай
- Суралцагч бүртгэх, төлбөр төлөх, ангид нэгдэх процесс гараар, төвөгтэй
- Эцэг эх хүүхдийнхээ ахиц, ирцийг хянах боломжгүй

## Манай шийдэл — Revo.edu
Revo.edu нь сургалтын төвүүдийг нэгдсэн платформ дээр нэгтгэж, сурагч, эцэг эх, сургалтын байгууллагуудыг ухаалаг системээр холбодог дижитал орчин юм.

## Гол боломжууд
- 🔎 **Сургалтын төв хайх** — Байршил, чиглэл, төлбөрөөр шүүн олох
- 💳 **Онлайн захиалга, төлбөр** — Шууд суудал захиалж, төлбөр төлөх
- 📅 **Календар, хуваарь** — Хичээлийн цагийг хянах
- 🧠 **Онлайн хичээл + Quiz тоглоом** — Сонирхолтой, мэдлэг бататгах тоглоомууд
- 💬 **Ангийн чат, мэдэгдэл** — Багш, сурагч, эцэг эхийн харилцаа
- ✅ **Ирцийн бүртгэл + эцэг эхийн хяналт** — Хүүхдийн оролцоог шууд харах
